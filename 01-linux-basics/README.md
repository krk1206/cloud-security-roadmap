#Linux Basics

이 폴더는 리눅스 기초 학습 내용을 정리한 공간이다.

## 목표
- 리눅스 기본 명령어 이해
- 권한과 프로세스 개념 익히기

---

## 실습환경
- os: WSL (Ubuntu 기반)
- shell: bash
- 학습 자료: 생활코딩 리눅스

---

## 핵심 개념 정리
- 다중  사용자 환경 지원
- 권한 기반 접근 제어
- 서버 운영에 최적화

---

## 사용자 & 권한
- root: 모든 권한을 가진 관리자
- 일반 사용자: 제한된 권한
- 파일은 사용자/그룹/기타로 권한이 나뉜다

---

## 프로세스 & 서비스
- 실행 중인 프로그램 = 프로세스
- 백그라운드에서 동작하는 프로세스 = 데몬

---

## 보안 관점 정리
- root 계정으로 상시 작업하는 것은 위험
- 불필요한 서비스는 공격 표면을 증가시킴
- 파일 권한 설정 오류는 권한 상승 취약점으로 이어질 수 있음

---

## 실습하면서 겪은 문제 & 해결
### 문제
- chmod 777의 위험성을 몰랐음

### 해결
- 최소 권한 원칙을 적용해야 함을 이해
