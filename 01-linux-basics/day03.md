## 오늘 공부한 내용
- 사용자 추가, 비밀번호 추가, sudo 권한 부여 (usermod -a -G sudo 사용자명)
- chmod를 이용한 실행 권한 부여
- 그룹 생성 및 그룹에 사용자 추가
- chown을 이용한 소유자 변경

## 이해한 것
- 사용자를 추가하더라도 (useradd) 비밀번호를 부여해야 접속 가능
- chmod 명령어로 다양한 방법 (8진법, o+r 등..)으로 파일에 rwx 권한 부여 가능
- chown 명령어로 파일, 디렉토리의 소유자, 소유 그룹 변경 가능 

## 헷갈린 것
- chmod에서 0~7까지의 숫자 숙지 미숙
 해결: x=1, w=2, r=4 인것을 외우고 나머지는 wx = 1+2 rwx = 1+2+4로 이해
- chown에서 파일 소유 그룹을 변경하더라도 권한을 부여하지 않으면 실행 불가
