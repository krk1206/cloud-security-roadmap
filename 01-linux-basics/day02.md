## 오늘 공부한 내용
- ps, locate, find, $PATH, alias, id, who 명령어 실습
- ctrl + z, &를 이용한 백그라운드 실행 및 fg를 이용하여 실행 (jobs로 확인)
- daemon
- cron을 이용하여 자동 실행
- root 계정으로 접속, root 권한으로 실행

## 이해한 것
- 서버 관리를 위한 명령어들로 윈도우의 작업관리자와 같은 역할 가능
- 백그라운드 실행을 이용하여 명령어 실행 대기시간 동안 다른 명령어 실행 가능
- daemon의 역할
- root 권한의 중요성, 위험성

## 헷갈린 것
- 백그라운드 실행 시 원하는 fg를 이용한 원하는 파일 열기
  (해결: jobs 명령어로 번호를 확인하고 %를 붙입니다.)
- $PATH를 이용한 명령어 실행
  (이해: 절대 경로(/bin/ls 등)를 다 치지 않고 파일 이름(ls)만 쳐도 실행되게 하는 지름길 목록)
