## 헷갈린 것 & 정리

### TCP (Transmission Control Protocol)
- 신뢰성이 중요한 전송에 사용
- 연결형 프로토콜
- 전송 순서 보장
- 수신 여부 확인 (ACK)
- 1:1 통신
- 신뢰성 높음
- 상대적으로 속도 느림

### UDP (User Datagram Protocol)
- 실시간성과 연속성이 중요한 서비스에 사용
- 비연결형 프로토콜
- 전송 순서 보장되지 않음
- 수신 여부 확인 없음
- 1:1 / 1:N / N:N 통신 가능
- 신뢰성 낮음
- 속도 빠름
