## 네트워크 식별자

| 구분 | 역할 | 특징 |
|------|------|------|
| MAC Address | 물리적 장비 식별 | 같은 네트워크 대역에서 사용 |
| IP Address | 논리적 위치 식별 | 네트워크 간 통신 |
| Port Number | 서비스 식별 | 하나의 IP에서 여러 서비스 구분 |

---

## OSI 7 Layer 개요
OSI 모델은 네트워크 통신을 기능별로 나눈 참조 모델이다.

### Application Layer (7)
- 사용자가 직접 접하는 계층
- HTTP, FTP, SMTP, DNS, DHCP 등
- 웹 브라우저, 메일 클라이언트 등과 연결됨

---

### Transport Layer (4)
- 송신자와 수신자 간 논리적 연결 담당
- 데이터 전송 신뢰성 보장 여부 결정
- 대표 프로토콜: TCP, UDP

---

### Network Layer (3)
- IP 주소 기반으로 목적지와 경로를 결정
- 라우팅 기능 수행

---

### Data Link Layer (2)
- 같은 네트워크 대역 내 통신 담당
- MAC Address 기반 전달
- 프레임 단위 전송 및 오류 검출

---

### Physical Layer (1)
- 전기적/물리적 신호 전송
- 케이블, 전압, 신호 규격 정의

---

## TCP/IP Protocol Stack

| TCP/IP 계층 | OSI 대응 | 주요 역할 |
|------------|----------|-----------|
| Application | 5~7 | 사용자 서비스 |
| Transport | 4 | TCP / UDP |
| Internet | 3 | IP, Routing |
| Network Access | 1~2 | MAC, Physical |

---

## TCP vs UDP

| 구분 | TCP | UDP |
|----|----|----|
| 연결 방식 | 연결형 | 비연결형 |
| 신뢰성 | 높음 | 낮음 |
| 순서 보장 | O | X |
| 수신 확인 | O | X |
| 속도 | 느림 | 빠름 |
| 사용 예 | 웹, 파일 전송 | 스트리밍, 게임 |

---

## 보안 관점 정리
- 포트는 외부에 노출되는 서비스 식별자이므로 관리 대상
- OSI 계층별로 발생하는 공격 유형이 다름
- TCP는 세션 하이재킹 공격 대상
- UDP는 반사/증폭 공격에 악용 가능
