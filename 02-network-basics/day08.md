## 오늘 공부한 내용
- ASCII 코드와 유니코드의 비트 단위 차이
- IP 주소의 2진수/10진수 표현 방식
- IPv4 클래스(A~E)의 용도 구분

## 이해한 것
- ASCII 코드는 7bit로 기본적인 영문자와 기호를 표현함
- 유니코드는 전 세계 문자를 표현하기 위해 8bit 또는 16bit를 사용하며,
  8bit는 1byte라는 기본 단위를 이룸
- IPv4 주소는 8bit 단위(옥텟) 4개로 구성되며,
  각 옥텟의 2진수를 사람이 보기 쉽게 10진수로 표현한 것임
- IPv4 클래스 중 A~C는 일반적인 공중망에서 사용되고,
  D 클래스는 멀티캐스트(IGMP), E 클래스는 연구·실험용으로 사용됨

## 헷갈린 것
- IPv4 클래스별 실제 사용 사례가 잘 떠오르지 않았음  
  (해결: A~C는 일반 통신, D는 멀티캐스트, E는 실사용 X로 큰 틀에서 정리)
